M3 PWA Shell Baseline

your knowledge, installable

This shell is the static, offline-capable container for the future chat experience. It already includes manifest metadata, service worker registration, and reserved surfaces for answer rendering and source attribution.

Install from a supported browser after Pages deployment to validate the standalone shell path for M3. Provider selection and author instance publication now come from static configuration files that can be shipped with the site.

Portable MCP work in youaskm3 is designed to remain source-aware and client-runnable.

Offline Status

  • Manifest linked from the served HTML shell.
  • Service worker registers and caches the shell assets.
  • Static assets stay compatible with GitHub Pages deployment.
  • Custom elements render the conversation and source surfaces.
  • Browser runtime module executes `search`, `remember`, `recall`, and `connect` locally.
  • Provider profiles and published author-instance metadata are loaded from static JSON.
